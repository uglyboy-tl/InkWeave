# 《琅琊榜》王权版设定

## 概述

基于《琅琊榜》IP 改编的王权类游戏。玩家扮演梅长苏（辅佐靖王），在朝堂博弈中平衡各方势力。游戏重心在于第二阶段（夺嫡布局），通过多线并行的策略推进，最终导向不同的同人文式结局。

---

## 四维属性设计

| 维度 | 对应原版 | 代表势力 | 核心逻辑 |
|------|----------|----------|----------|
| **圣眷** | Church (教会) | 梁帝、皇室 | 皇帝的信任与猜忌。夺嫡的根本，也是最大的威胁。 |
| **朝望** | People (民众) | 六部文臣、世家大族 | 政令推行的基础。太低被弹劾，太高被指结党营私。 |
| **兵权** | Army (军队) | 禁军、边防军、赤焰旧部 | 自保与翻案的底牌。太低任人宰割，太高被指谋反。 |
| **盟势** | Money (财富) | 江左盟、江湖势力、天泉山庄 | 情报网与暗线。太低失去耳目，太高江湖干政引朝廷忌惮。 |

## 失败文案 (Game Over)

每个维度都有**过低**和**过高**两种死法，贴合原著基调：

### 1. 圣眷 (Imperial Favor)
- **过低 (≤0)**: 梁帝疑心大起，一纸诏书削去爵位。
  *"朕的江山，容不得你算计。"*
  赐鸩酒，薨。
- **过高 (≥10)**: 功高震主，鸟尽弓藏。
  梁帝冷笑：*"祁王如此，你亦如此。"*
  暗中毒杀，死因成谜。

### 2. 朝望 (Court Prestige)
- **过低 (≤0)**: 六部联名弹劾，御史台死谏。
  *"靖王不尊君父，不敬朝纲！"*
  废为庶人，圈禁至死。
- **过高 (≥10)**: 世家门阀尾大不掉，朝堂尽入私囊。
  文臣反客为主，皇权旁落，天下大乱。

### 3. 兵权 (Military Power)
- **过低 (≤0)**: 谢玉派兵围剿，禁军无力回天。
  *"麒麟才子，终究算漏了人心。"*
  死于乱军，赤焰冤案永埋。
- **过高 (≥10)**: 拥兵自重，被指谋逆。
  *"清君侧"*之名传遍天下，背负千古骂名，众叛亲离。

### 4. 盟势 (Alliance Influence)
- **过低 (≤0)**: 江左盟分崩离析，暗线尽断。
  夏江冷笑：*"梅长苏，你的江湖到头了。"*
  悬镜司介入，真相永无天日。
- **过高 (≥10)**: 江湖干政，名门正派群起攻之。
  *"武林盟主也敢染指朝堂？"*
  朝廷与江湖联手剿灭，江左盟覆灭。

## 核心事件链

剧情由以下核心事件驱动。阶段二包含四条并行事件线，全部完成后进入阶段三（终局）。

### 阶段一：潜龙入渊 (铺垫期)
*目标：梅长苏入京，通过小案引起注意，促成靖王回京结盟。*

| 事件名 | 依赖事件 | 剧情作用 |
| :--- | :--- | :--- |
| **1. 苏哲入京** | 无 | 游戏开始。梅长苏化名入京，各方势力开始接触。 |
| **2. 兰园惊变** | 1 | 发现枯井尸骨，牵连户部。梅长苏初显身手，引起靖王注意。 |
| **3. 靖王回京** | 2 | 靖王从北境回京，正式入局。梅长苏成为其谋士，夺嫡开始。 |

### 阶段二：夺嫡风云 (核心玩法)
*目标：削弱太子与悬镜司势力。四条线**必须全部完成**，才能触发高潮。*

| 事件线 | 事件名 | 依赖事件 | 剧情作用 |
| :--- | :--- | :--- | :--- |
| **朝堂线**<br>(瓦解太子党) | **4. 户部洗牌** | 3 | 沈追上位，掌控钱粮命脉。 |
| | **5. 谢玉倒台** | 4 | 景睿生日宴揭露罪行，太子党瓦解。 |
| **悬镜线**<br>(拔除夏江) | **6. 卫峥被抓** | 3 | 夏江发难，离间靖王与梅长苏。 |
| | **7. 查封悬镜司** | 6 | 成功营救，夏江伏法，身份半公开。 |
| **誉王线**<br>(压制九王) | **8. 誉王结盟** | 3 | 九王爷暗中布局，拉拢言侯与庆历军。 |
| | **9. 掌控庆历军** | 8 | 切断誉王兵源，化解谋反隐患。 |
| **江湖后宫线**<br>(稳固后方) | **10. 越贵妃失势** | 3 | 破解后宫迷香局，打压越氏。 |
| | **11. 霓凰结盟** | 10 | 南境安定，江左盟与云南王府联动。 |

### 阶段三：终局 (同人文结局)
*目标：应对最终危机，翻案昭雪。此阶段不再追求数值平衡，而是**关键抉择**，结合第二阶段的积累，导向不同结局。*

| 事件名 | 依赖事件 | 剧情作用 |
| :--- | :--- | :--- |
| **12. 九安叛乱** | **5 且 7 且 9 且 11** | 誉王狗急跳墙，起兵谋反。四线全部解决，才有实力应对。 |
| **13. 血战勤王** | 12 | 靖王率军救驾。确立储君地位，圣眷达到顶峰。 |
| **14. 身份危机** | 13 | 靖王得知梅长苏就是林殊。**关键抉择点**：隐瞒还是坦白？ |
| **15. 金殿鸣冤** | 14 | 最终翻案。根据之前的标志位和选择，触发不同结局。 |

## 多结局设计 (同人文风格)

结局不再依赖复杂的数值计算，而是由**"第二阶段的关键标志位"** + **"第三阶段的抉择"** 组合而成。

| 结局名称 | 触发条件示例 | 结局文案示例 |
| :--- | :--- | :--- |
| **海晏河清 (True End)** | `身份危机`选择坦白 + 双线完美通关 | 靖王理解并支持梅长苏，君臣相得。金殿鸣冤成功，沉冤昭雪，新帝登基。梅长苏选择留在朝堂辅佐新君。"这大梁的江山，我们一起守。" |
| **孤臣末路 (Bad End A)** | `身份危机`选择隐瞒 + 盟势过高 | 翻案成功，但梅长苏因隐瞒身份心力交瘁，毒发身亡。靖王孤独登基，终身怀念。"世间再无林殊，只有大梁天子。" |
| **功败垂成 (Bad End B)** | `金殿鸣冤`时圣眷过低 | 梁帝翻脸，下旨赐死梅长苏。靖王抗旨被废，赤焰案永埋地下。"麒麟之才，终究算漏了人心。" |
| **众叛亲离 (Bad End C)** | `金殿鸣冤`时兵权过高 | 靖王拥兵自重，被指谋逆。梅长苏百口莫辩，死于乱军之中。"清君侧之名传遍天下，背负千古骂名。" |
| **江湖逍遥 (Hidden End)** | `身份危机`选择坦白 + 盟势极高 | 翻案成功后，梅长苏选择归隐江湖，与霓凰相伴。"林殊已死，梅长苏也该走了。" |

---

## 设计思路

1. **重心在第二阶段**：玩家的主要精力放在四线并行的夺嫡布局上，积累不同的"隐性标志位"（如是否保住了卫峥、是否彻底得罪梁帝等）。
2. **第三阶段是"收束器"**：不再增加新维度，而是根据第二阶段的积累和第三阶段的抉择，快速导向结局。
3. **同人文风格**：结局文案强调"如果当初..."的解读感，让玩家觉得"这确实可能发生"。
4. **重玩价值**：不同的选择组合导致不同的结局，鼓励玩家多周目尝试。
