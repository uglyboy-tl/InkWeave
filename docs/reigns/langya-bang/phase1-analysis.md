# 阶段一分析：梅长苏入京（动线梳理版）

本文档专门分析《琅琊榜》王权版第一阶段（梅长苏入京）的场景、人物与流程节点。此阶段的核心特征是**“蛰伏与铺垫”**。梅长苏入京时已自带“麒麟才子”虚名，需通过**积累声望**与**暗中调查**，解锁关键人脉。

**核心双轨机制：**
1.  **明线（声望/社交）**：通过市井、妙音坊、贵族社交积累名声。声望达到低阈值即可解锁蒙挚，达到高阈值解锁高层权贵。**注意：声望获取途径有限，均为一次性收益。**
2.  **暗线（线索/调查）**：通过兰园探查、苏宅分析积累线索。线索积累配合声望，解锁关键盟友沈追。

## 1. 流程节点 (Flow)

流程卡用于强制引导剧情，作为新手教程熟悉四维属性与各方势力，并顺势开启主线。

| 节点名称 | 出现时机 | 作用/维度 | 选项与数值影响 | 后续逻辑 |
| :--- | :--- | :--- | :--- | :--- |
| **琅琊阁榜** | 游戏首次启动 | **盟势**教学。蔺晨发布榜单，江湖暗流涌动。 | 左：顺势造势 (`+1M`) / 右：低调避嫌 (`-1M`) | 进入 `梁帝侧目` |
| **梁帝侧目** | 节点 1 后 | **圣眷**教学。榜单传入宫中，皇室态度试探。 | 左：坦然受之 (`+1C`) / 右：称病避嫌 (`-1C`) | 进入 `北境军情` |
| **北境军情** | 节点 2 后 | **兵权**教学。旧部与军方反应，北境暗线呼应。 | 左：暗中联络 (`+1A`) / 右：隐匿踪迹 (`-1A`) | 进入 `清流议论` |
| **清流议论** | 节点 3 后 | **朝望**教学。京城文臣热议，**有人邀约梅长苏进京**。 | 左：受邀进京 (`+1P`) / 右：秘密进京 (`-1P`) | 触发 `苏哲入京` 事件 |

## 2. 场景动线梳理 (Scene Flow)

### 2.1 苏宅内室
- **初始状态**：✅ 初始常驻
- **解锁条件**：无
- **核心功能**：情报中枢、状态调节。黎纲在此汇报动向，提供基础情报与四维微调途径。
- **包含卡片**：
  - `黎纲汇报`：黎纲送来各方动向，需你定夺。 -> 整理情报 / 调息服药
  - `病发调息`：旧疾发作，身体虚弱，需紧急处理。 -> 服用乌金丸 / 运功压制
  - `密信处理`：收到江左盟或北境传来的密信。 -> 亲自拆解 / 交由黎纲处理

### 2.2 市井茶楼
- **初始状态**：✅ 初始常驻
- **解锁条件**：无
- **核心功能**：社会反响收集、四维微调、双线索收集枢纽。
- **动态调整**：
  - `沈追解锁` -> 添加“听到兰园传闻”选项（交互后解锁 `兰园废墟` 场景，开启 `兰园线索` 收集）。
  - `沈追解锁` -> 添加“户部贪腐”相关选项（开启 `贪腐线索` 收集）。
- **包含卡片**：
  - `市井流言`：茶楼酒肆议论纷纷，暗藏线索。 -> 暗中查访 / 不予理会
  - `麒麟热议`：众人议论琅琊阁榜单，对你的入京充满好奇。 -> 放任传播 / 派人压制
  - `兰园异闻`（沈追解锁后）：听说兰园废弃多年，近日传出井中异响。 -> 派人探查（指定下一张为 `兰园废墟`） / 暂不理会（将 `兰园废墟` 添加到卡池）
  - `户部风波`（沈追解锁后）：茶客议论户部账目混乱，尚书贪腐。 -> 暗中记录（获得贪腐线索） / 转移话题

### 2.3 兰园废墟
- **初始状态**：❌ 锁定
- **解锁条件**：`市井茶楼交互（听到兰园传闻）`
- **核心功能**：核心线索源、暗线布局起点。探查枯井、移尸设局，直接推动线索积累。
- **动态调整**：
  - `初始探查/移尸卡交互` -> 替换为“阴森/异动氛围卡”。
  - `阴森/异动氛围卡交互` -> 永久移除。
- **包含卡片**：
  - `兰园废墟`：来到兰园废墟，枯井深不见底，似有古怪。 -> 枯井探查（获得兰园线索） / 移尸布局（获得兰园线索）
  - `阴森异动`（替换卡）：枯井周围气氛诡异，似有暗流涌动。 -> 暗中监视 / 散布谣言

### 2.4 妙音坊
- **初始状态**：✅ 初始常驻
- **解锁条件**：无
- **核心功能**：暗线枢纽、声望辅助（一次性）、宫羽线入口。监听、密会、抚琴传谣。
- **动态调整**：
  - `初访妙音坊交互` -> 触发事件，解锁宫羽人物卡。
  - `宫羽解锁后` -> 移除 `初访妙音坊`。
- **包含卡片**：
  - `初访妙音坊`：初次来到妙音坊，宫羽抚琴迎客，曲中暗藏玄机。 -> 抚琴传谣（一次性声望） / 静静听琴

### 2.5 户部衙门
- **初始状态**：❌ 锁定
- **解锁条件**：沈追人物卡解锁后
- **核心功能**：线索加速器、户部专属操作。提供高阶线索获取途径与账目核查选项。
- **动态调整**：
  - `沈追解锁` -> 添加至卡池。
- **包含卡片**：
  - `户部查账`：沈追邀请查看户部旧账，发现多处涂改痕迹。 -> 仔细核对（获得贪腐线索）/ 询问沈追（无贪腐线索）
  - `证据封存`：收集到关键贪腐证据，需决定存放地点。 -> 移交沈追（获得贪腐线索）/ 藏于苏宅（无贪腐线索）

### 2.6 贵族府邸
- **初始状态**：❌ 锁定
- **解锁条件**：首次完成 `萧景睿 & 言豫津` 触发卡交互
- **核心功能**：高阶声望加速器（一次性）、高层社交入口。提供一次性高收益声望事件（雅集/骑射）。
- **动态调整**：
  - `萧景睿 & 言豫津交互成功` -> 添加至卡池。
  - `一次性声望事件触发` -> 权重骤降或移除。
- **包含卡片**：
  - `贵族雅集`：萧景睿设宴，京城名流云集。 -> 吟诗作对（+一次性声望）/ 品茗论道（+一次性声望）
  - `骑射邀约`：言豫津邀你去校场骑马射箭。 -> 下场比试（+一次性声望）/ 旁观喝彩（+一次性声望）

## 3. 人物动线梳理 (Character Flow)

### 3.1 谢玉
- **初始状态**：✅ 初始常驻
- **解锁条件**：无
- **核心功能**：政治试探、早期压力源。代表太子党/宁国侯府的拉拢与威慑，测试玩家政治倾向。
- **动态调整**：
  - `2 次拒绝谢玉送礼` -> 触发“谢玉警告”事件，**指定下一张卡片为 `谢玉警告`**。
  - `触发谢玉警告` -> **阻止后续太子人物卡加入**（若声望达标时已触发此事件，则太子不再出现）。
- **包含卡片**：
  - `谢玉送礼`：谢玉送来厚礼，暗示结盟。 -> 欣然收下 / 婉言谢绝
  - `谢玉警告`：谢玉派人送来警告信，语气不善。 -> 强硬回击 / 虚与委蛇

### 3.2 飞流
- **初始状态**：✅ 初始常驻
- **解锁条件**：无
- **核心功能**：武力调节、趣味缓冲、盟势微调。保护宗主、日常互动、处理突发武力事件。
- **动态调整**：
  - `特定危机事件` -> 添加 `飞流抓刺客` 卡片。
- **包含卡片**：
  - `飞流捣乱`：飞流在院子里练武或抢零食，打断思考。 -> 陪他切磋 / 给他买零食
  - `飞流抓刺客`（危机触发后）：飞流发现屋顶有黑衣人影。 -> 活捉审问 / 直接击退

### 3.3 太子
- **初始状态**：❌ 锁定
- **解锁条件**：`声望 >= 1` 且 `未触发谢玉警告`
- **核心功能**：东宫拉拢、政治施压。代表太子党利益，通过谢玉等代理人进行试探。
- **动态调整**：
  - `声望 >= 1 且 未触发谢玉警告` -> 添加 `太子招揽` 卡片。
  - `玩家拒绝拉拢` -> 触发“东宫施压”事件，**指定下一张卡片为 `东宫施压`**。
- **包含卡片**：
  - `太子招揽`：太子派使者送来密信，许诺高位。 -> 接受暗示 / 保持距离
  - `东宫施压`（拒绝后）：太子党对你的中立态度不满，暗中施压。 -> 正面回击 / 暂避锋芒

### 3.4 誉王
- **初始状态**：✅ 初始常驻
- **解锁条件**：无
- **核心功能**：七珠亲王拉拢、利益许诺。代表另一股夺嫡势力，态度相对温和但暗藏算计。
- **动态调整**：
  - `玩家拒绝拉拢` -> 触发“誉王监视”事件，**指定下一张卡片为 `誉王监视`**。
- **包含卡片**：
  - `誉王拜访`：誉王亲自到访，许诺六部之位拉拢。 -> 虚与委蛇 / 严词拒绝
  - `誉王监视`（拒绝后）：发现誉王派人在苏宅附近徘徊。 -> 引蛇出洞 / 闭门不见

### 3.5 萧景睿 & 言豫津
- **初始状态**：✅ 初始（作为**触发卡**存在）
- **解锁条件**：无（初始入池，但仅抽中一次即转化）
- **核心功能**：社交钥匙、场景解锁器。引荐进入高层圈子，解锁 `贵族府邸` 场景。
- **动态调整**：
  - `交互成功` -> 移除触发卡，永久解锁 `贵族府邸` 场景，添加更多人物交互卡进入卡池。
- **包含卡片**：
  - `萧言邀约`：萧景睿与言豫津邀请你同游京城。 -> 欣然赴约 / 改日再聚

### 3.6 宫羽
- **初始状态**：❌ 锁定
- **解锁条件**：`妙音坊` 场景交互触发特定事件后解锁
- **核心功能**：暗线执行者。监听、传递密报。
- **动态调整**：
  - `妙音坊场景交互触发事件` -> 添加宫羽人物卡至卡池。
- **包含卡片**：
  - `妙音坊听琴`：宫羽抚琴，曲中暗藏玄机。 -> 抚琴传谣（一次性声望）/ 静静听琴（修正四维）
  - `暗线情报`：宫羽送来江左盟暗探收集的密报。 -> 采纳建议 / 暂缓行动

### 3.7 蒙挚
- **初始状态**：❌ 锁定
- **解锁条件**：`声望 >= 1`
- **核心功能**：军方纽带、靖王伏笔、兵权调节。闻风夜访，确认身份，传递北境消息。
- **动态调整**：
  - `声望 >= 1` -> 添加 `蒙挚夜访` 卡片。
  - `蒙挚夜访交互` -> **移除该卡片**，添加常驻交互卡（如 `飞流切磋`）。
- **包含卡片**：
  - `蒙挚夜访`（一次性）：禁军大统领深夜叩门，询问北境旧部近况。 -> 透露风声 / 含糊其辞
  - `飞流切磋`（常驻）：蒙挚见猎心喜，与飞流过招。 -> 旁观喝彩 / 出手试探

### 3.8 沈追
- **初始状态**：❌ 锁定
- **解锁条件**：`声望 >= 1`
- **核心功能**：核心剧情推手、户部线入口、双线索收集起点。慕名而来，提供查账途径与贪腐线索。
- **动态调整**：
  - `声望 >= 1` -> 添加 `沈追请托` 卡片。
  - `沈追请托交互` -> **移除该卡片**，添加户部账册等专属卡组，同步解锁 `户部衙门` 场景。
  - `沈追解锁后` -> 触发 `市井茶楼` 添加“兰园传闻”与“户部贪腐”相关选项，开启双线索收集途径。
- **包含卡片**：
  - `沈追请托`（一次性）：户部侍郎沈追深夜来访，言辞恳切。 -> 公开建言 / 避嫌自保

## 4. 阶段推进逻辑总结 (双线索收集)

1.  **初始状态**：
    *   流程卡 `琅琊阁榜` 炸出名声，蔺晨离场。
    *   卡池注入：苏宅、市井、妙音坊（场景） + 谢玉、飞流、誉王、萧/言触发卡（人物）。
2.  **声望积累与太子线判定（明线）**：
    *   玩家通过 `宫羽（抚琴传谣）` 或 `贵族府邸（雅集/骑射）` 获取 **声望**（一次性收益）。
    *   **声望 >= 1** -> 解锁 `蒙挚` 与 `沈追`。
    *   **太子线分支判定**：
        *   若 `未触发谢玉警告` -> 添加 `太子` 人物卡至卡池（开启东宫拉拢线）。
        *   若 `已触发谢玉警告` -> **不添加** `太子` 人物卡（太子线关闭，玩家需独自面对谢玉施压）。
3.  **双线索开启（暗线）**：
    *   `沈追解锁` -> `市井茶楼` 新增“兰园传闻”与“户部贪腐”选项。
    *   交互“兰园传闻” -> **解锁 `兰园废墟` 场景**，开启 `兰园线索` 收集。
    *   交互“户部贪腐” / `沈追人物卡` / `户部衙门` -> 开启 `贪腐线索` 收集。
4.  **阶段一核心目标**：
    *   玩家交替推进，分别积累 `兰园线索` 与 `贪腐线索`。
    *   当 **双线索均达到阈值** -> 触发阶段一核心事件（枯井现尸），宣告阶段一结束，进入第二阶段。
